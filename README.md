# ⚡ JavaScript em 7 Dias + Projetos Reais

> *"Crie Aplicações Web, Interaja com o DOM, Consuma APIs e Desenvolva Projetos Reais para seu Portfólio."*

Este repositório contém todo o código, exercícios práticos e projetos desenvolvidos ao longo do curso **JavaScript em 7 Dias**, cobrindo desde os fundamentos básicos até Programação Orientada a Objetos e consumo de APIs assíncronas.

---

## 🎯 Objetivos de Aprendizado

- [x] **Fundamentos & Lógica:** Variáveis, operadores, condicionais (`if/else`), estruturas de repetição e funções.
- [x] **Estrutura de Dados:** Arrays, Objetos, manipulação de Strings e métodos avançados.
- [x] **DOM & Eventos:** Manipulação da árvore DOM, escuta de eventos e interatividade em tempo real.
- [x] **APIs & Asincronismo:** Requisições HTTP, `fetch`, Promises e sintaxe `async/await`.
- [x] **POO (Programação Orientada a Objetos):** Classes, protótipos, herança e encapsulamento.
- [x] **Boas Práticas:** Padrões de código utilizados no dia a dia do mercado.

---

## 🛠️ Tecnologias Utilizadas

- **JavaScript (ES6+)**
- **HTML5 & CSS3**
- **Git & GitHub** (Versionamento e deploy)
